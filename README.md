# SEL_PrácticaPro

Proyecto desarrollado en el marco de la **Práctica Profesional de Ingeniería en Informática**, orientado al análisis, diseño y desarrollo de una **plataforma de gestión de servicios de transporte** para la empresa **Servicios El Loa**.

## Objetivo del proyecto
Diseñar e implementar una solución informática que permita digitalizar y optimizar la gestión de servicios de transporte, considerando distintos tipos de servicios, clientes y vehículos, aplicando buenas prácticas de ingeniería de software, modelado de datos y arquitectura.

## Alcance
El proyecto contempla:
- Levantamiento y análisis de requerimientos.
- Diseño de arquitectura del sistema.
- Modelado de base de datos relacional normalizada.
- Desarrollo de backend y frontend.
- Integración con base de datos mediante contenedores Docker.
- Documentación técnica y bitácora de práctica profesional.

## Tecnologías consideradas
- **Backend:** Node.js / API REST  
- **Frontend:** React  
- **Base de datos:** PostgreSQL  
- **Contenedores:** Docker  
- **Modelado:** ERD normalizado  
- **Control de versiones:** Git y GitHub
